# 利用激光雷达数据辅助风机功率预测

## 项目背景

峡沙56号风机机舱上架设了一台**激光雷达（LiDAR）**，可以测量风机前方不同距离处（如 100 m、150 m、200 m 等）的气象特征（水平风速 HWS、垂直风速 VWS、风向等）。与此同时，56 号风机自身通过 **SCADA** 系统记录风速、风向、有功功率等运行数据，南瑞（**NARI**）平台也对同一台风机的数据进行采集和导出。

本项目旨在利用激光雷达的前瞻性测量信息，探索其对风机功率预测的辅助作用，主要研究问题包括：

- 激光雷达数据是否能提升风机功率预测精度？
- 哪些距离下的激光雷达数据最有参考价值？
- 不同预测时长下，激光雷达数据的辅助效果有何差异？
- 哪些特征变量（如 HWS、VWS、TI 等）对功率预测贡献最大？

## 数据说明

### 时间范围

| 数据集 | 时间范围 | 时间粒度 |
|--------|---------|---------|
| SCADA 有功功率 | 2024 年全年 | 1 分钟 |
| NARI 平台数据 | 2024-08-30 ~ 2024-11-20 | 1 分钟 |
| 激光雷达数据 | 与风机数据对应 | 10 分钟 |

> **注意**：RAW_DATA 和 PROCESS_DATA 中的部分文件仅为截取的片段，实际完整数据行数更多。

### SCADA 与 NARI 时间戳问题

经互相关分析发现，南瑞（NARI）平台导出数据相对于 SCADA 数据存在**系统性时间戳滞后**，且随时间段变化：

| 时段 | NARI 滞后量 |
|------|-----------|
| 2024-08-30 ~ 2024-09-30 | +2 分钟 |
| 2024-10-01 ~ 2024-11-07 17:59 | +3 分钟 |
| 2024-11-07 18:00 ~ 2024-11-20 | +4 分钟 |

分段变步长对齐后，Pearson 相关系数从 0.9767 提升至 **0.9957**，RMSE 从 449.22 kW 降低至 **193.54 kW**。
详见 [峡沙56号_SCADA与南瑞数据时间戳对齐分析报告.md](./峡沙56号_SCADA与南瑞数据时间戳对齐分析报告.md)。

## 运行说明


按脚本编号顺序依次执行 `CODE/` 目录下的脚本：

```bash

python "#2-1对齐风机导出数据和南瑞数据.py"
python "#2-2DASH对比风机数据和南瑞数据.py"    # 启动 Dash 服务，浏览器查看
python "#3时间粒度1变10.py"
python "#4风机-激光雷达数据合并.py"
python "#5风速功率散点图DASH.py"              # 启动 Dash 服务，浏览器查看
python "#6统计不同距离数据的有效率.py"
python "#7提取所有距离数据功率同时有效.py"
```

### 单独运行时间戳对齐脚本

```bash
# 使用默认路径（推荐）
python align_timestamps.py

python align_timestamps.py \
    --input  DATA/峡沙56号_合并风机导出数据和南瑞数据.csv \
    --output DATA/峡沙56号_时间戳对齐后数据.csv
```
