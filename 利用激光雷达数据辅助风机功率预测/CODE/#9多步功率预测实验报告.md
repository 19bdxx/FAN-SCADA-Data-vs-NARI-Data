## 一、版本演进

| 版本 | 变更 |
|-----|------|
| v1 | 5 场景（S1/S2a~S2d），仅 90m vs 全距离，无气象特征 |
| v2 | 21 场景，三层实验设计（距离扫描+气象增益+去SCADA），补全气象特征提取 |
| **v3（本版）** | **在层1补充"无SCADA"对比组（N1/N2/N3/N4），共 34 场景，回答"LiDAR与SCADA是否相互干扰"** |

**v3 的研究动机**：v2 层1结果显示 LiDAR+SCADA（M1~M5）整体不如纯 SCADA（M0），但无法判断原因：  
- 假设 A：**LiDAR 信息本身对多步预测贡献不足**  
- 假设 B：**LiDAR 风速与 SCADA 机舱风速之间存在信号干扰**，混合使用反而互相抵消  

若假设 B 成立，则去掉 SCADA 后仅使用 LiDAR 的效果应该好于 M1（LiDAR+SCADA），甚至可能超越 M0。

## 二、三层实验设计（第三版）

```
层1（28 个场景）── 风速距离结构 + 无SCADA对比

  含 SCADA 组（M 组，原15个）：
    M0  : 仅 SCADA 机舱风速 + 历史功率（基准）
    M1  : 单距离 LiDAR（40, 60, ..., 300m）+ SCADA + 功率（10个）
    M2  : 近距组（40~120m LiDAR HWS）+ SCADA + 功率
    M3  : 中距组（150~210m LiDAR HWS）+ SCADA + 功率
    M4  : 远距组（240~300m LiDAR HWS）+ SCADA + 功率
    M5  : 全距离（40~300m LiDAR HWS）+ SCADA + 功率

  无 SCADA 组（N 组，新增13个）：回答"SCADA 是否干扰 LiDAR？"
    N1  : 单距离 LiDAR（40, 60, ..., 300m）+ 功率，不含SCADA（10个）
    N2  : 近距组（40~120m LiDAR HWS）+ 功率，不含SCADA
    N3  : 中距组（150~210m LiDAR HWS）+ 功率，不含SCADA
    N4  : 远距组（240~300m LiDAR HWS）+ 功率，不含SCADA
    注：N5（全距离无SCADA）= 层3 L3a，已涵盖

层2（4 个场景）── 气象特征增益实验（同 v2，M5 作为 E0 基准）
    E1  : M5 + VShear；E2 : M5 + HShear；E3 : M5 + TI；E4 : M5 + 全气象

层3（2 个场景）── 去 SCADA 验证（同 v2）
    L3a : 仅全距离 LiDAR HWS + 功率（无SCADA）
    L3b : 仅全距离 LiDAR HWS + 全气象 + 功率（无SCADA）
```

**建模**：LightGBM / XGBoost 全部 34 场景；LSTM 仅 4 关键场景（M0/M5/L3a/L3b）  
**时间设置**：`LOOK_BACK=6`（60 min 历史），`FORECAST_STEPS=[1,2,3,6]`（+10~+60min）

## 三、层1 结果 A：无SCADA（N1）vs 含SCADA（M1）——SCADA 干扰量分析

### 3.1 Δ RMSE = N1_RMSE（无SCADA）− M1_RMSE（含SCADA）（LightGBM）

> **符号解读**：Δ > 0 = 去掉 SCADA 后变差（SCADA 有益）；Δ < 0 = 去掉 SCADA 后变好（**SCADA 有害**）

| 距离 | +10min M1/N1/Δ | +20min M1/N1/Δ | +30min M1/N1/Δ | +60min M1/N1/Δ |
|-----|--------------|--------------|--------------|--------------|
| 40m | 442/447/**+5** | 626/639/**+13** | 754/755/+1 | 966/980/+14 |
| 60m | 444/447/**+4** | 630/638/**+8** | 756/766/**+10** | 960/972/**+12** |
| 90m | 446/452/**+7** | 629/639/**+10** | 763/756/**-7** | 969/979/+10 |
| 120m | 445/448/**+3** | 634/637/**+3** | 758/759/+1 | 981/974/**-7** |
| 150m | 446/442/**-4** | 630/633/+3 | 762/753/**-8** | 976/966/**-10** |
| 180m | 444/442/**-3** | 635/637/+1 | 765/763/**-2** | 977/975/-2 |
| 210m | 443/442/**-1** | 633/634/+1 | 764/755/**-8** | 973/971/-2 |
| 240m | 443/439/**-4** | 632/634/+3 | 764/753/**-11** | 979/973/**-7** |
| 270m | 445/438/**-6** | 629/630/+1 | 752/746/**-6** | 961/969/+8 |
| 300m | 447/440/**-6** | 631/629/**-3** | 752/748/**-5** | 968/974/+5 |

**粗体/负数 = 去掉 SCADA 后更好（SCADA 对该距离造成干扰）**

### 3.2 关键发现 A：SCADA 干扰具有距离依赖性

| 距离带 | +10min | +20min | +30min | +60min | 结论 |
|-------|--------|--------|--------|--------|------|
| 近距（40~120m） | SCADA 有益（Δ>0） | SCADA 有益 | 混合 | 多为有益 | 近距 LiDAR 与 SCADA 风速同向，SCADA 提供独立增量 |
| 中远距（150~300m） | **SCADA 干扰（Δ<0）** | 接近零 | **SCADA 干扰** | 混合 | **中远距 LiDAR 风速与机舱风速高度相关，混合后引入噪声** |

**物理解释**：激光雷达 150~300m 处测量的是与轮毂高度相近的上游自由流风速，与 SCADA 机舱测风具有高度相关性但并不完全一致（机舱测风受尾流和叶片扰动影响）。混合使用时，模型难以区分两路冗余信号，导致正则化失效，性能反而下降。近距（40~120m）的激光雷达风速受建筑/湍流影响更大，与 SCADA 独立性更高，因此 SCADA 提供的是独立增量信息，两者互补。

### 3.3 关键发现 B：**单独 LiDAR（N1，150~300m）在 +10min 步长上超越纯 SCADA（M0）**

> M0 SCADA 基准（LightGBM）+10min = **443.8 kW**

| 场景 | +10min RMSE | 与M0比 | 评注 |
|-----|------------|--------|------|
| N1_270m_only | **438.4** | **-5.4 kW** ✅ | 仅用 270m LiDAR HWS，超越 SCADA 基准 |
| N1_240m_only | **439.3** | **-4.4 kW** ✅ | |
| N1_300m_only | **440.4** | **-3.3 kW** ✅ | |
| N1_180m_only | **441.6** | **-2.2 kW** ✅ | |
| N1_210m_only | **442.2** | **-1.6 kW** ✅ | |
| N1_150m_only | **442.0** | **-1.7 kW** ✅ | |
| M0: 仅SCADA | 443.8 | 基准 | |
| M1_270m: LiDAR+SCADA | 444.6 | +0.8 kW | 加了 SCADA 反而变差 |

**这推翻了假设 A，验证了假设 B**：问题不是"LiDAR 信息不足"，而是"中远距 LiDAR 与 SCADA 相互干扰"。  
**单独使用中远距 LiDAR（去掉 SCADA）在短步长上能达到最佳效果。**

## 九、七项关键结论（v3 更新）

| # | 结论 | 证据 |
|--|------|------|
| 1 | **LiDAR+SCADA 效果差的主因是"信号干扰"，不是"LiDAR信息不足"** | N1（仅LiDAR）中远距在+10min超越M0（仅SCADA），而M1（LiDAR+SCADA）未超越 |
| 2 | **中远距 LiDAR（150~300m）与 SCADA 机舱风速存在冗余干扰** | Δ RMSE 在 150~300m 大多为负（去SCADA后更好），在 40~120m 多为正 |
| 3 | **仅中距 LiDAR（N3，150-210m）在全部步长上超越所有含SCADA方案** | N3+10min=438.8，优于M0(443.8)、M3(444.0)、M5(447.4) |
| 4 | **仅远距 LiDAR 单距离（N1_270m）在+10min上是最佳单特征配置** | N1_270m +10min=438.4，超 M0(443.8) 5.4kW |
| 5 | **多步 LSTM 全面优于树模型**（与 #8 单步结论相反），LSTM M0 为全局最优 | LSTM M0 vs LightGBM N3 |
| 6 | **气象特征在多步框架下无明显增益，TI 有害** | E3 +60min RMSE = 1016（+34kW vs E0） |
| 7 | **LSTM 下去SCADA（L3a）在+10min优于LiDAR+SCADA（M5）** | L3a=446 < M5=461 |
