## 二、三层实验设计（第四版）

```
层1（29 个场景）── 风速距离结构

  功率自回归基准（P 组，新增）：
    P0  : 仅历史功率（无任何风速）————最弱基准，下界参考

  含 SCADA 组（M 组，原15个）：
    M0  : 仅 SCADA 机舱风速 + 历史功率
    M1  : 单距离 LiDAR（40~300m）+ SCADA + 功率（10个）
    M2/M3/M4/M5 : 近/中/远/全距离 LiDAR + SCADA + 功率

  无 SCADA 组（N 组，原13个）：
    N1  : 单距离 LiDAR（40~300m）+ 功率（10个）
    N2/N3/N4 : 近/中/远距离 LiDAR + 功率

层2（4 个场景）── 气象特征增益（E1~E4，基于 M5）
层3（2 个场景）── 去 SCADA 验证（L3a/L3b）
```

**建模**：LightGBM + XGBoost + GBM（HistGBM）全部 35 场景；LSTM 仅 4 关键场景（M0/M5/L3a/L3b）  
**时间设置**：`LOOK_BACK=6`（60 min 历史），`FORECAST_STEPS=[1,2,3,6]`（+10~+60min）

## 三、层1 核心发现：P0（仅历史功率）是树模型最优基线

### 3.1 P0 vs M0 vs N3 — 三模型完整对比（RMSE, kW）

| 场景 | 模型 | +10min | +20min | +30min | +60min | 平均 |

| M0: 仅SCADA | LightGBM | 444 | 623 | 744 | 941 | 688 |
| N3: 仅中距LiDAR | LightGBM | 439 | 633 | 756 | 970 | 699 |
| | | | | | | |
| **P0: 仅历史功率** | **XGBoost** | **432** | **599** | **706** | **907** | — |
| M0: 仅SCADA | XGBoost | 431 | 608 | 729 | 931 | — |
| N3: 仅中距LiDAR | XGBoost | 443 | 628 | 753 | 969 | — |
| | | | | | | |
| P0: 仅历史功率 | GBM | 430 | 610 | 713 | 916 | — |
| M0: 仅SCADA | GBM | 432 | 611 | **709** | 916 | — |
| N3: 仅中距LiDAR | GBM | 432 | 615 | 715 | 928 | — |

### 3.2 LightGBM 层1全场景排名（+10min 步长）

| 排名 | 场景 | +10min RMSE |
|-----|------|------------|
| **1** | **P0: 仅历史功率** | **432.3** |
| 2 | N1_270m（仅LiDAR 270m） | 438.4 |
| 3 | N3_mid（仅中距LiDAR） | 438.8 |
| 4 | N1_240m / N1_300m | 439~440 |
| ... | ... | ... |
| 14 | **M0: 仅SCADA** | **443.8** |

**P0 以 432.3kW 排名第一，领先 M0（443.8kW）11.5kW，领先 M5（447.4kW）15.1kW。**

### 3.3 关键发现：纯功率自回归击败所有风速方案（LightGBM/XGBoost）

> 平均 RMSE 全步长排名（LightGBM，层1场景，RMSE 由小到大）：

| 排名 | 场景 | 平均RMSE |
|-----|------|---------|
| **1** | **P0: 仅历史功率** | **673.2** |
| 2 | M0: 仅SCADA | 688.1 |
| 3 | N1_270m_only | 695.8 |
| 4 | M1_270m | 696.6 |
| 5 | M1_40m | 697.0 |
| 6 | M1_60m | 697.4 |
| 7 | N1_300m_only | 697.6 |

**P0 在所有 29 个层1场景中平均 RMSE 最低。**

**物理解释**：在 60 min 历史回望窗口下，前几步的历史功率序列已充分编码了近期风速变化信息（功率与风速立方成正比，功率历史即是风速历史的高阶映射）。额外引入风速特征在 6×10min 的时序结构中不能提供净增量，反而增加了特征噪声与维度负担。

## 四、层1 发现的修订（结合 P0）

v3 的结论"LiDAR 与 SCADA 相互干扰"仍然成立，但现在有了更强的参照系：

| 问题 | v3 结论 | v4 修订 |
|-----|---------|---------|
| 为什么 LiDAR+SCADA 效果差？ | LiDAR 与 SCADA 冗余干扰 | ✅ 仍成立，但根本原因更深：**任何风速特征在多步时序框架下对树模型都无净增益** |
| 最优策略 | N3（仅中距LiDAR） | **P0（仅历史功率）≻ M0 ≻ N3**，无风速最优 |
| SCADA 干扰量 | N1 中远距 < M1（去SCADA后更好）| ✅ 仍成立，但 P0 比 N1 更优（完全去除风速更好）|

**新的模型鲁棒性验证**：LightGBM 和 XGBoost 均显示 P0 最优；GBM（HistGBM）在 +30/+60min 中 M0 或 L3a 与 P0 接近，差异在 3kW 以内，总体结论一致。

## 五、三模型一致性验证（P0/M0/N3/L3a）

| 场景 | LightGBM+10 | XGBoost+10 | GBM+10 | 三模型一致性 |
|-----|------------|-----------|--------|------------|
| P0: 仅历史功率 | **432** | **432** | **430** | ✅ 三模型均最优 |
| M0: 仅SCADA | 444 | 431 | 432 | ✅ 均次优 |
| N3: 仅中距LiDAR | 439 | 443 | 432 | ✅ 均中等 |
| L3a: 仅全距LiDAR | 449 | 442 | **426** | ⚠️ GBM例外（426最好） |
| M5: 全距+SCADA | 447 | 438 | 428 | ✅ 均差于P0 |

> ⚠️ **GBM 例外**：GBM（HistGBM）在 L3a（全距离LiDAR，无SCADA）上 +10min=426，是 GBM 模型下的最优。GBM 对高维特征（10个HWS）的正则化能力更强，因此全距离 LiDAR 在 GBM 下的表现优于 LightGBM/XGBoost。

## 八、九项关键结论（v4 更新）

| # | 结论 | 证据 |
|--|------|------|
| 1 | **P0（仅历史功率）在树模型下是层1全局最优** | LightGBM 平均RMSE=673，领先所有风速场景 |
| 2 | **任何风速特征在60min历史窗口+树模型下对多步预测均无净增益** | P0 < M0 < M1~M5（LightGBM/XGBoost均一致） |
| 3 | **GBM（HistGBM）在高维特征场景（L3a/L3b）表现优于LightGBM/XGBoost** | GBM L3a=426，GBM L3b=423，超越P0 |
| 4 | **LiDAR与SCADA干扰结论仍成立**（v3结论），但根因更深：连添加单一风速（M0）都不如P0 | M0 > P0（RMSE更高）|
| 5 | **三GBM模型结论高度一致**（除L3a/L3b GBM例外），证明结论具有模型鲁棒性 | P0最优在LightGBM/XGBoost中均成立 |
| 6 | **LSTM仍是全局最优**（M0+LSTM=427kW），体现时序建模能力 | LSTM M0 < 树模型P0 |
| 7 | **中远距LiDAR（N1，150-300m）去SCADA后超越M0**（v3结论不变）| N1_270m=438 < M0=444（LightGBM） |
| 8 | **气象特征（TI）对树模型有害，VShear在+30/+60min有微弱增益** | 三模型一致 |
| 9 | **GBM下L3b（仅LiDAR HWS+气象，无SCADA）=423kW为树模型最优** | 气象特征在GBM中轻度有益 |

## 九、后续建议（v4）

| 建议 | 方向 |
|-----|------|
| **LSTM + P0** | 在 P0 场景跑 LSTM，验证是否在树模型和 LSTM 下 P0 都最优 |
| **更长历史回望** | P0 依赖历史功率，增加 `LOOK_BACK=12`（120 min）测试 P0 上限 |

| **特征重要性分析** | 分析 M0/M5 中各历史步长和风速特征的重要性 |
